1. Cada jogador coloca uma peça de cada vez no tabuleiro até completar o total das 24 peças.

2. O jogador da vez movimenta suas peças para um espaço vazio nos sentidos para cima, para baixo ou para os lados, mas não nas diagonais.

3. Após um movimento, se o jogador conseguir ter três peças numa linha, ele captura uma peça do seu adversário, isto é, remove uma peça do adversário à sua escolha.

4. Os jogadores não podem ter mais de três peças numa linha a qualquer momento. Uma linha feita no início do jogo, quando as peças estão a ser posicionadas, não vale. Apenas uma peça por vez pode ser removida por jogada, mesmo que tenham sido feitas mais do que uma linha com apenas um movimento.

5. Quando um jogador não pode mais fazer linhas ou fazer jogadas válidas, perde o jogo.
