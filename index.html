1. Cada jogador posiciona uma peça de cada vez no tabuleiro até completar o total das 24 peças.

2. O jogador da vez movimenta suas peças para um espaço vazio nos sentidos para cima, para baixo ou para os lados, mas não nas diagonais.

3. Quando um jogador consegue posicionar três peças em uma linha, ele elimina uma peça de seu adversário. Isso é conhecido como “comer” o inimigo.

4. Os jogadores não podem ter mais de três peças em uma linha a qualquer momento. Uma linha feita no início do jogo, quando as peças estão sendo posicionadas, não vale. Somente uma peça por vez pode ser removida do jogo, mesmo que tenham sido feitas mais de uma linha com apenas um movimento.

5. Quando um jogador não pode mais fazer linhas, perde o jogo.
